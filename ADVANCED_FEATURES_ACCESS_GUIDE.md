## 🎯 **How to Access Each Feature**

### **1. Multi-Language Support**
1. **Look for**: Language switcher (flag icon) in top-right
2. **Click**: Dropdown to see language options
3. **Select**: English 🇺🇸, Tamil 🇮🇳, or Hindi 🇮🇳
4. **Result**: Entire UI changes to selected language instantly

### **2. Video Consultation**
1. **Go to**: Patient dashboard
2. **Click**: "Video" tab (📹 icon)
3. **Current Status**: Shows placeholder with appointment requirement
4. **Future**: Will integrate with appointment booking for direct video calls

### **3. Pharmacy Integration**
1. **Go to**: Patient dashboard
2. **Click**: "Pharmacy" tab (🏪 icon)
3. **Features Available**:
   - Search medicines by name
   - Filter by location/city
   - See real-time availability
   - Compare prices
   - Contact pharmacies directly
   - Get directions

### **4. Offline Records**
1. **Look for**: Offline indicator in bottom-right corner
2. **Status**: Shows "Online/Offline" with sync status
3. **Features**:
   - Automatic data sync when online
   - Manual sync button
   - Shows pending items count
   - Works without internet connection
